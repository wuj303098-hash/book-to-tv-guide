# Book to TV Guide

Static SEO guide site for romance shows and book adaptations, launching with an Every Year After content cluster.

## Pages

- Home guide index
- Every Year After topic hub
- Season 2 status
- Ending explained
- Charlie cliffhanger guide
- Percy secret explainer
- Cast and characters
- Book vs show changes
- Episode guide
- About, privacy, and contact pages

## Commands

```bash
npm run generate
npm test
npm run build
```

The build command regenerates the static pages and verifies SEO structure, sitemap coverage, canonical tags, structured data, and content depth.

Set `SITE_URL=https://your-domain.com` before running the build when a custom domain is ready.
